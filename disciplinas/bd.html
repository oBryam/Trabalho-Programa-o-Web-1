Componente Curricular: Banco de Dados
Carga Horária total: 72 h C.H. Extensão: 00 h Período Letivo: 5º semestre
Ementa
Introdução aos Sistemas de Gerenciamento de Bancos de Dados (SGBD), características, usos e vantagens. Tipos/evolução de SGBDs; Modelagem Conceitual: Modelo Entidade-Relacionamento, técnicas de modelagem. Modelo Relacional: Conceitos, Normalização, Álgebra Relacional. Introdução ao SQL através de um SGBD.
